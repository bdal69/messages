Messages

A JavaScript app to send a message to several services at once.



About Messages


Messages is a JavaScript app created during a labfab project.


Send you message:

Send a message to Twitter, Facebook, Slack and Gmail at once. Messages sent will be logged on the server in the "messages_sent_log.txt" file.

Type you message( a maximum off 140 caracteres are allowed to meet Twitter requirement).
